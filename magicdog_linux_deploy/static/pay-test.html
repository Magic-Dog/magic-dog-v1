HTML集成代码 / HTML Integration Code

复制以下代码到您的HTML页面中 / Copy the following code to your HTML page:

HTML 
<!-- 样式文件引入 -->
<link rel="stylesheet" href="http://{Project URL}/static/style.css">

<!-- 支付插件脚本配置 -->
<script id="darcula-js"
    app="#payment-button-container,#wc_payment_methods"
    color="#26222f"
    page="credit-payment?submitButton=submit&lang=en"
    api-base-url="{Project URL}"
    scale="1"
    show-scale-panel="false"
    template_type="自定义type"
    src="http://{Project URL}/static/darcula.js">
</script>
故障排除 / Troubleshooting
样式问题： 如果页面样式异常，请导入Bootstrap和FontAwesome CDN资源
脚本错误： 检查服务器地址是否正确，确保网络连接正常

配置参数详解
app
支付表单注入容器ID
color
支付按钮主题颜色
page
支付完成后跳转页面 不填写使用网站默认跳转
api-base-url
API服务器基础地址
scale
表单UI缩放比例
show-scale-panel
是否显示缩放面板
template_type
自定义表单定义 不填写则使用默认
快速开始
1 替换 {Project URL} 为服务器地址
2 确保目标容器元素存在
3 将代码添加到 </body> 前
4 刷新页面，支付表单自动加载
